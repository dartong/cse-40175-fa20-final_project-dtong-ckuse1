# cse-40175-fa20-final_project-dtong-ckuse1
Final project for Ethical and Professional Issues, Fall 2020 semester.

This is a choose-your-adventure (CYA) text-based game, where the player follows different perspectives in a story of a whistleblower, his superior, and the media.
